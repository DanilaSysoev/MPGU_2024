1. Создать класс Logger, осуществляющий логирование информации в абстрактный приемник сообщений.
При создании объекта логгера ему передается указатель на приемник сообщений.
Класс должен содержать метод log с одним текстовым аргументом, записывающий переданный аргумент
в приемник.

2. Реализовать приемник сообщений, который пишет сообщения в конец заданного файла.

3. Добавить уровни сообщений (например INFO, DEBUG, WARNING, ERROR, CITICAL).
Реализовать возможность создать логгер, записывающий только сообщения заданного уровня и выше.
Интерфейс класса продумайте сами.

4. Создайте тесты логгера, использующие мок для приемника сообщений, проверяющие функциональность логгера.
