1. Создайте класс Frame со следующими свойствами:
    - позиция на экране
    - размеры

2. Добавьте в класс Frame следующие события (экземпляры Event из задания по OOP от 241202)
    - OnCursorEnter - срабатывает, когда курсор пересекает границу фрейма снаружи->внутрь
    - OnCursorExit - срабатывает, когда курсор пересекает границу фрейма изнутри->наружу
    - OnClick - срабатывает, когда курсор над фреймом и происходит нажатие левой кнопки мыши

3. Добавьте в класс Frame методы:
    - Update - обновляет состояние фрейма, вызывает события
    - Draw - отрисовывает фрейм. Настройки отрисовки выберите по своему усмотрению.

4. Создайте по одному обработчику событий (дочерние классы для EventHandler) для тестирования фрейма.
   Достаточно будет обработки события через вывод сообщения в консоль о том, что событие произошло.
   Для созданных EventHandler и Event добавьте необходимые реализации EventArgs EventArgsBuilder.
