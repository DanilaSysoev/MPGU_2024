1. Создать класс Entity с произвольным набором полей (2-3 штуки). Создать к ним get-аксессоры
2. Создать класс Repository с массивом (вектором) сущностей. Добавить метод get(i) доступа по индексу i к сущности
3. Добавить в данный класс виртуальный метод connect, иммитирующий подключение к БД. Метод должен выводить строку "Connect to <db engine>",
   где <db engine> - ваш любимый движок БД
4. Создать класс UnlovedRepository, наследующий Repository и переопределяющий connect. Метод должен выводить строку "Connect to <db engine>",
   где <db engine> - ваш НЕлюбимый движок БД.
5. В main продемонстрировать работу программы.
